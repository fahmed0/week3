Flightscape London

This project involved fetching flight data through an API, preprocessing the data, and applying various machine learning models to predict flight prices. After retrieving the data, one-hot encoding, standardization, and data splitting were implemented to prepare it for supervised learning algorithms such as Linear Regression, Lasso Regression, Ridge Regression, and Decision Trees. The Decision Trees model emerged as the best-performing model for predicting flight prices based on the given dataset.

Customer Clustering

Utilising customer data from an automobile company, I applied four unsupervised machine learning algorithms (KMeans, DBSCAN, Agglomerative Clustering, and Gaussian Mixture Model) to identify distinct customer segments. After evaluating model performance using Silhouette Score, Calinski-Harabasz Index, and Davies-Bouldin Index, DBSCAN emerged as the best choice for creating well-separated and dissimilar clusters. This customer segmentation enables targeted marketing strategies and better understanding of customer behaviour in the new market.

Poisonous Mushrooms Classification

In the Mushroom Classification project, supervised machine learning algorithms were applied to accurately identify poisonous and edible mushrooms based on their characteristics. Utilising Python libraries such as NumPy, Pandas, Matplotlib, and Scikit-learn, the project involved data preprocessing, feature engineering, and model evaluation. By comparing the performance of different supervised classification models, an optimal solution was developed to safeguard public health and elevate the overall mushroom foraging experience.
